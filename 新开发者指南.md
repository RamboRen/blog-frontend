## 💡 常见问题

### Q1: 我应该先读哪个文档？

**A:** 根据你的角色选择：
- **前端** → `COMPLETE_TUTORIAL.md` → 前台开发教程
- **后端** → `后端开发手册.md` → 完整阅读
- **全栈** → 两个都要读，先看教程再看手册

### Q2: 环境搭建遇到问题怎么办？

**A:** 
1. 查看 `COMPLETE_TUTORIAL.md` 第 3 章
2. 检查 Node.js 版本（>= 18.x）
3. 尝试清理缓存：`npm cache clean --force`
4. 使用备用方案：`npm install --legacy-peer-deps`

### Q3: 如何连接真实后端 API？

**A:** 
1. 阅读 `后端开发手册.md` 第 4 章
2. 参考 `admin/src/services/api.ts` 的 Mock 实现
3. 修改 API_BASE_URL 为你的后端地址
4. 添加认证 Token 支持

### Q4: 如何部署到生产环境？

**A:** 
1. 阅读 `COMPLETE_TUTORIAL.md` 第 8 章
2. 前端推荐 Vercel（一键部署）
3. 后台可部署到任意 Node.js 服务器
4. 参考 Docker 配置进行容器化部署
