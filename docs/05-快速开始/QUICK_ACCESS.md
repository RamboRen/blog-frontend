## 🐛 已知问题排查

### 如果遇到问题

#### 1. 页面无法访问
```bash
# 检查 admin 项目是否启动
cd /Users/renbo/workspace/blog-frontend/admin
npm run dev
```

#### 2. 样式不正常
```bash
# 重新安装依赖
cd admin
rm -rf node_modules package-lock.json
npm install --legacy-peer-deps
```

#### 3. 路由错误
确保访问的是正确的路径：
- `/tags` 不是 `/tag`
- `/categories` 不是 `/category`

#### 4. 弹窗不显示
- 检查浏览器控制台是否有错误
- 清除浏览器缓存
- 刷新页面
