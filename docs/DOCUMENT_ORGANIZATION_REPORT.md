## 📝 文档管理规范

### 新增文档流程

1. **确定分类**：选择合适的子目录
2. **命名文件**：使用大写 + 下划线格式
3. **更新索引**：在 `docs/README.md` 中添加链接
4. **提交代码**：Git 提交并说明用途

### 文档命名规范

```
✅ 正确：ABOUT_PAGE_SUMMARY.md
✅ 正确：DARK_THEME_COMPLETE.md
❌ 错误：AboutPage.md
❌ 错误：dark-theme.md
```

### 目录命名规范

```
✅ 正确：01-项目概述
✅ 正确：02-功能实现
❌ 错误：project_overview
❌ 错误：功能
```
