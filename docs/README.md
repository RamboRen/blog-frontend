## 📝 文档维护

### 文档位置
所有文档统一存放在 `docs/` 目录下，按类别分子目录。

### 命名规范
- 文件名使用大写字母和下划线：`FEATURE_NAME.md`
- 目录名使用中文 + 数字前缀：`01-项目概述`

### 更新流程
1. 新增文档时，选择合适的分类目录
2. 如果现有分类不合适，可创建新目录
3. 更新本文档的索引信息
